## üìä How to Use the Data

The data is structured in a simple JSON format, which is easy to navigate. Here‚Äôs a quick guide on understanding the contents:

- **FUND_NAME:** The name of the fund.
- **JSE_TICKER:** The specific ticker used on the Johannesburg Stock Exchange.
- **CATEGORY:** Type of product (e.g., ETF, ETN).
- **DESCRIPTION:** A brief overview of what the product offers.
- **DIVIDEND_FREQUENCY:** Information on how often dividends are paid.

You can read the data in any JSON-compatible application. If you're using it for analysis or development, consider exploring libraries that help parse JSON data.
